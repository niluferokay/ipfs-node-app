# ipfs-node-app
A simple IPFS web app with Node.js to upload a document to IPFS and view it on the browser.

# Installation
1.Download the app
```
git clone https://github.com/niluferokay/ipfs-node-app.git
cd ipfs-node-app
```
2.Install required packages
```
npm install 
```
3.Start the app
```
ipfs daemon
node app.js
```
